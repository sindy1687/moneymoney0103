# 記帳應用 - 主輸入畫面

一個手機優先的記帳應用主輸入介面，採用可愛插畫風格，支援分類選擇和數字鍵盤輸入。

## 功能特色

- ✅ **手機優先設計** - 專為手機螢幕優化
- ✅ **SVG 圖標設計** - 精美的自定義 SVG 圖標，帶紫色輪廓
- ✅ **頂部狀態欄** - 顯示時間和網絡狀態
- ✅ **Header 標籤** - 支出/收入/轉帳三種模式切換
- ✅ **分類先選 → 再輸入金額** - 直覺的操作流程
- ✅ **單頁完成記帳** - 無需跳轉頁面
- ✅ **底部固定數字鍵盤** - 方便快速輸入，運算符在左側
- ✅ **即時計算** - 支援加減乘除運算
- ✅ **分類管理** - 推薦/未分組/設定三個Tab

## 使用方法

1. 直接在瀏覽器打開 `index.html`
2. 選擇一個分類（例如：飲食）
3. 使用數字鍵盤輸入金額
4. （可選）輸入備註
5. 點擊「✔️ 儲存」完成記帳

## 分類列表

**第一行：**
- 娛樂 - 3D眼鏡圖標
- 美髮/修眉 - 剪刀和梳子圖標
- 醫療 - 急救箱圖標

**第二行：**

- 寵物 - 狗和貓頭圖標
- 旅行 - 海灘傘圖標

**第三行：**
- 存股 - 錢袋圖標
- 媽媽菜錢 - 食物圖標
- 太太礦錢 - 手拿硬幣圖標
- 美甲 - 指甲圖標

**第四行：**
- 養老金 - 手拿硬幣圖標

## 資料儲存

記帳資料儲存在瀏覽器的 `localStorage` 中，鍵名為 `accountingRecords`。
